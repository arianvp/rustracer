# Rustracer
Rustracer is an efficient GPU Path tracer written in Rust and Vulkan

ACTIVE DEVELOPMENT IS ON THE `nee-temp` BRANCH

## BVH

BVH Implementation lives as a fork here https://github.com/arianvp/bvh/commits/master
